1）关闭防火请

如果看到有很多信息，就需要使用 stop 命令关闭防火墙。

如果要彻底关闭防火墙的话可以输入：

Linux chkconfig命令用于检查，设置系统的各种服务。

这是Red Hat公司遵循GPL规则所开发的程序，它可查询操作系统在每一个执行等级中会执行哪些系统服务，其中包括各类常驻服务。

这样重启 Linux 后防火墙就会被彻底关闭。

然后还要关闭 selinux

如果安装 MySQL 需要对 MySQL 进行卸载，如果在安装 Linux 时没有选择最小化安装，可能会自带 MySQL，不过版本很低。
