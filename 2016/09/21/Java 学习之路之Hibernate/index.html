Java 学习之路之Hibernate

##前言

公司让我转岗，从 iOS 转向 Java，于是开始进行 Java 的学习，由于是带有目的性的学习所以学习速度上要快一些，所以只要能将功能实现就 ok 了。至于深入那是以后的事情，目前需要走的东西就是胜任工作。

##Hibernate简介

Hibernate 是在 JDBC 层面上的一层封装实现的 ORM（即Object-Relational Mapping，它的作用就是在关系型数据库和对象之间做了一个映射。从对象（Object）映射到关系（Relation），再从关系映射到对象。这样，我们在操作数据库的时候，不需要再去和复杂SQL打交道，只要像操作对象一样操作它就可以了（把关系数据库的字段在内存中映射成对象的属性））
